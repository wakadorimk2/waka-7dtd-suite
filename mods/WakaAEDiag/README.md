# Waka AE Diagnostic v0.1

`perkAdvancedEngineering` の effect_group integrity を検証するための診断 logging mod。

開発・デバッグ専用、本番 modlist では無効化推奨。

## 機能

`perkAdvancedEngineering` の effect_group XML 構造をログ出力し、他 mod の override 後の最終状態を可視化する。XPath パッチが衝突して効果が消えていないか確認する用途。

## 必要環境

- 7 Days to Die v2.6
- EAC OK（pure XML）

## インストール

`mods/WakaAEDiag/` を MO2 経由でインストール、または 7DTD の `Mods/WakaAEDiag/` に配置。
