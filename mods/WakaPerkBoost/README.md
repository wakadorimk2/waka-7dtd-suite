# Waka Perk Boost v0.1.0

[Black Wolf's better vanilla perks](https://www.nexusmods.com/7daystodie/mods/4067) のコンパニオン。Bloodfall - Hardcore Overhaul のキツい進行を補正する。

## 解決する問題

Bloodfall は：

- `xp_to_level` が伸びる
- perk costs が上がる
- armor LootProb が抑制される
- max_level 622

これと Black Wolf perks を素直に組み合わせると、perk 1 段の効果が「上げる労力に見合わない」感じになる。

## 機能

Black Wolf の戦闘ダメージ・サバイバルレジスト・looter スケーリング系を**全体的に約 +50%** リフト。Bloodfall 込みで「上げた perk が効いてる」感を取り戻す。

加えて、各 rank の long description を Bloodfall 風の散文で書き直し。「上げた甲斐がある」記述に変える（メタ的だが、判定の重みは変わる）。

## 必要環境

- 7 Days to Die v2.6
- Black Wolf's better vanilla perks v9.3+
- Bloodfall - Hardcore Overhaul v2.0+
- EAC OK（pure XML）

## インストール

`mods/WakaPerkBoost/` を MO2 経由でインストール。Black Wolf と Bloodfall より**後**にロードすること。
